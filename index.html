📁 연락처 파일 업로드 사용법

✨ 100명도 10초만에 등록!

아이폰, 안드로이드, 맥북에서 연락처를 VCF 파일로 내보낸 후
업로드하면 모든 정보가 자동으로 입력됩니다!

📱 아이폰/아이패드에서 내보내기

방법 1: iCloud 사용 (추천) ⭐
1️⃣ 맥/PC에서 iCloud.com 접속
• iCloud.com → 로그인
• [연락처] 클릭

2️⃣ 연락처 선택
• 좌측 하단 [모든 연락처] 클릭
• 또는 개별 연락처 선택 (Cmd+클릭으로 여러 개)

3️⃣ vCard 내보내기
• 좌측 하단 ⚙️ (설정) 아이콘 클릭
• [vCard 내보내기...] 선택
• VCF 파일 다운로드 완료!

4️⃣ CRM에 업로드
• CRM [📁 연락처 파일 업로드] 클릭
• 다운로드한 VCF 파일 선택
• 자동 등록 완료! 🎉
방법 2: 연락처 앱 사용
1️⃣ 연락처 공유
• 연락처 앱 열기
• 연락처 선택
• [연락처 공유] 탭
• [파일로 저장]

2️⃣ 파일 앱에서 찾기
• 파일 앱 열기
• iCloud Drive 또는 다운로드 폴더
• .vcf 파일 확인

3️⃣ AirDrop으로 전송
• 맥으로 AirDrop
• 또는 이메일로 자신에게 전송

🤖 안드로이드에서 내보내기

1️⃣ 연락처 앱 열기

2️⃣ 메뉴 → 설정
• 우측 상단 ⋮ (메뉴)
• [설정] 또는 [관리]

3️⃣ 연락처 내보내기
• [연락처 내보내기] 또는 [Export]
• VCF 파일 형식 선택
• 저장 위치 선택

4️⃣ 파일 찾아 업로드
• 다운로드 폴더에서 VCF 파일 확인
• CRM에 업로드

💻 맥북에서 내보내기

1️⃣ 연락처 앱 열기

2️⃣ 연락처 선택
• Cmd+A: 전체 선택
• Cmd+클릭: 개별 선택

3️⃣ 내보내기
• 파일 메뉴 → [vCard 내보내기...]
• 저장 위치 선택
• VCF 파일 생성 완료!

4️⃣ CRM에 업로드
• [📁 연락처 파일 업로드] 클릭
• 저장한 VCF 파일 선택

iCloud 자동 동기화 설정

iPhone, iPad, Mac에서 데이터를 자동으로 동기화합니다

🎯 작동 방식

  • 데이터 변경 시 3초 후 자동으로 iCloud에 저장
  • 다른 기기에서 열 때 자동으로 최신 데이터 로드
  • 충돌 발생 시 자동으로 최신 버전 선택
  • 오프라인에서도 작동 (온라인 시 동기화)
📱 Safari 브라우저 필수
iCloud 동기화는 Safari에서만 작동합니다.
Chrome, Edge에서는 지원되지 않습니다.
